Kook Boards

Kook Boards is a React surfing Ecommerce website built on the MERN stack. This was a final group project in our web development cohort built by three developers. Our website allows a user to purchase different surfing accessories and checkout their order via Stripe. My key contributions to this site:

  • Setting up user registration and login functionality and connecting the user model to MongoDB
  • Using JWT and Bcrypt for security and authentication
  • Decoding the user on the front end to get user information for checkout and order history

ShredLightly

ShredLightly is a frontend only website created by three developers to help users find bike trails in a city of their choice. The user inputs a city and bike trails within a 50 mile radius populate. My key contributions to this website were:

  • Javascript DOM maniulations
  • API fetches to RapidAPI and the Google maps API
  • Assiting with CSS styling
Art Auction

The art auction website was created by three developers using NodeJS and Express. This project was backend focused with emphasis on CRUD operations. This website allows a user to register and log in with an account. From there you are able to browse art pieces up for auction and place a bid. My key contributions to this app were:

  • Setting up the login functionality with Bcrypt and JSONwebtokens
  • Moving throughout the app as a logged in user and secured Admin routes
  • The submit bid function that saves users bids to the database
